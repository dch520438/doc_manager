# 文档管家

一款基于 Python + Tkinter 的**智能文件分类与全文搜索工具**，适用于 Windows 和麒麟（Linux）系统。完全离线运行，数据安全可控。

## 功能特性

### 智能文件整理
- **自定义规则分类** — 根据文件名关键词、扩展名自动将文件分类到指定文件夹
- **一键整理桌面** — 批量整理桌面散落文件
- **实时监控** — 开启后自动检测桌面新文件并按规则归类
- **自动创建文件夹** — 目标文件夹不存在时自动创建

### 全文搜索
- **已管理文档搜索** — 搜索已整理归档的文档
- **全盘搜索** — 扫描全电脑所有磁盘，按文件名或文件内容搜索
- **指定路径搜索** — 自定义搜索范围，支持多路径（分号分隔）
- **实时显示结果** — 搜索过程中实时显示已找到的文件，支持随时停止
- **内容搜索** — 深入文档内部提取文本进行关键词匹配

### 支持的文件格式

| 类型 | 格式 |
|------|------|
| Word | `.doc` `.docx` `.wps` |
| Excel | `.xls` `.xlsx` `.csv` `.et` `.ett` |
| PDF | `.pdf` |
| PPT | `.ppt` `.pptx` `.dps` |
| 文本 | `.txt` `.md` `.log` `.rtf` |
| 标记语言 | `.xml` `.json` `.html` `.htm` |

### 内容提取能力（三层保障）

| 层级 | 方式 | 说明 |
|------|------|------|
| 第一层 | 专用库提取 | python-docx、openpyxl、pdfplumber 等（最佳质量） |
| 第二层 | ZIP/OLE 解析 | 无需第三方库，直接解析 docx/xlsx/pptx/et/doc/wps 内部结构 |
| 第三层 | 二进制扫描 | 通用文本提取，支持 UTF-8/GBK/GB2312/Big5 等多种编码 |

> 即使未安装任何第三方依赖，程序仍可通过内置备用方案提取大部分文档的文本内容。

### 其他功能
- **重建索引** — 对已有文档重新提取内容，修复搜索不到的问题
- **依赖诊断** — 在设置中查看各依赖库的安装状态，测试文件内容提取效果
- **文件操作** — 打开文件、打开所在文件夹、删除记录

## 系统要求

- Python 3.8+
- Windows 10/11 或 麒麟/UOS 等 Linux 发行版
- tkinter（Python 自带，通常无需额外安装）

## 快速开始

### Windows

```bash
# 方式一：双击启动（推荐）
双击 install.bat    # 自动安装依赖并启动
# 或
双击 启动文档管家.bat

# 方式二：命令行
cd doc_manager
pip install -r requirements.txt
python main.py
```

### 麒麟 / Linux

```bash
# 方式一：启动脚本（推荐，自动检测安装依赖）
chmod +x start.sh
./start.sh

# 方式二：手动安装
cd doc_manager
pip3 install -r requirements.txt --break-system-packages
python3 main.py
```

## 依赖说明

| 依赖包 | 用途 | 必需 |
|--------|------|------|
| `python-docx` | Word .docx 内容提取 | 推荐 |
| `openpyxl` | Excel .xlsx 内容提取 | 推荐 |
| `pdfplumber` | PDF 内容提取（中文效果好） | 推荐 |
| `PyPDF2` | PDF 基础内容提取 | 推荐 |
| `python-pptx` | PPT .pptx 内容提取 | 推荐 |
| `olefile` | 旧版 .doc/.wps/.et 内容提取 | 推荐 |
| `chardet` | 编码检测（增强通用提取） | 推荐 |
| `ttkbootstrap` | UI 美化 | 可选 |

> 所有依赖均为**推荐安装**，未安装时程序仍可运行，使用内置备用方案提取内容。

## 使用说明

### 1. 创建分类规则
- 点击左侧「管理规则」→「添加规则」
- 设置规则名称、包含关键词、目标文件夹
- 可设置优先级和是否启用

### 2. 整理文件
- **一键整理** — 点击顶部「一键整理桌面」按钮
- **实时监控** — 点击「实时监控」按钮，开启后自动整理桌面新文件

### 3. 搜索文件
- 在左侧搜索框输入关键词
- 选择搜索范围：
  - **已管理** — 搜索已整理归档的文档
  - **全盘** — 搜索全电脑所有磁盘
  - **指定路径** — 输入路径或点击「浏览」选择文件夹
- 勾选「搜索文档内容」可按文件内容搜索
- 点击「搜索」按钮开始

### 4. 重建索引
- 如果搜索不到已有文档的内容，点击顶部「重建索引」按钮
- 程序会重新提取所有已管理文档的文本内容

### 5. 诊断问题
- 点击「设置」→ 查看「内容提取依赖状态」
- 使用「测试文件内容提取」功能，选择文件测试提取效果

## 安全说明

- **完全离线运行** — 不联网、不上传、不追踪
- **数据本地存储** — 使用 SQLite 数据库，文件保存在程序目录
- **不修改文件内容** — 文件整理仅移动/复制，不改变文件内容
- **开源透明** — 所有代码可审查
